# Readable Numbers

The **Readable Numbers** ROUNDS mod is a mod that make number slightly more readable for **Tabinfo**, **Infoholic**, and **Root's Stats UI**.
For "large" number like **1,000,000**, it will display as **1M**.

## Controls

- **Left Control** - Toggle between "Subbfix" and "Name" number formatting.
- **Right Control** - Toggle off number formatting, until you release the button.

## Mod Compatibility

- **[Tabinfo](https://thunderstore.io/c/rounds/p/willuwontu/TabInfo/)** - This mod works with the **Tabinfo** mod to make numbers more readable in the Tabinfo window.
- **[Infoholic](https://thunderstore.io/c/rounds/p/Penial/Infoholic/)** - This mod works with the **Infoholic** mod to make numbers more readable in the Infoholic window.
- **[Root's Stats UI](https://thunderstore.io/c/rounds/p/Root/Root_Stats_UI/)** - This mod works with the Infoholic mod to make numbers more readable in the **Root's Stats UI** window.
