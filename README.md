# Readable Numbers

The **Readable Numbers** ROUNDS mod is a mod that make number slightly more readable for **Tabinfo** and **Infoholic**.
For "large" number like 1,000,000, it will display as 1M.

## Controls

- **Left Control** - Toggle between "Subbfix" and "Name" number formatting.
- **Right Control** - Toggle off number formatting, until you release the button.

## Mod Compatibility

- **Tabinfo** - This mod works with the Tabinfo mod to make numbers more readable in the Tabinfo window.
- **Infoholic** - This mod works with the Infoholic mod to make numbers more readable in the Infoholic window.
